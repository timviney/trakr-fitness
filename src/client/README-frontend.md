# Trakr Frontend (Vue 3 + TypeScript + Vite)

Quick start:

1. cd into the client folder:

   ```bash
   cd src/client
   ```

2. Install dependencies:

   ```bash
   npm install
   # or: pnpm install
   ```

3. Start development server:

   ```bash
   npm run dev
   ```

Available scripts:
- `dev` — start Vite dev server
- `build` — build for production
- `preview` — preview built site
- `type-check` — run `vue-tsc` to check types
